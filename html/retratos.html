RETRATOS


¿CÓMO CREAR UN RETRATO?
  1. Busca una imagen para dibujar
  2. Traza el boceto para tener la base del dibujo
  3. Agraga los elementos base del dibujo
  4. Crea las sombras
  5. Crea la iluminacion
  6. Agrega detalle al dibujo


¿QUÉ MATERIALES USAR?
  • Lápiz
  • Borrador
  • Sacapuntas
  • Plumas
  • Corrector
  • Hojas blancas
